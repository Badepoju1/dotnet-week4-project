# Student Management System

**Author:** Babatunde Adepoju

## Project Description
The Student Management System is a simple console-based application written in C#. It allows users to manage student records by adding, viewing, updating, and deleting student information. The system also calculates statistics such as average grade, highest and lowest grades, and counts of passing and failing students.  

**Features include:**
- Add new students with ID, name, and grade
- View all student records
- Calculate average grade
- Search for students by ID
- Update student grades
- Delete student records
- Display grade statistics (highest, lowest, pass/fail counts)
